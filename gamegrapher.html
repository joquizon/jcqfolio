GAMEGRAPHER
context:
Front End Development Class proj. 2015


The Gamegrapher project was created to illustrate the use of a search API and the visual translation of the data received from the search. The project is a web application that when queried recieves and graphs scores from major video gaming review sites. It is currently live and can be visited here.
