THE BAUER COUNT
context:
Information Graphics class proj. 2014


The Bauer Count project was designed for a current event infographic assignment. Created to celebrate the return of Fox's 24 to the small screen, the project was designed to show the number of on-screenkills Jack Bauer had in the original 8 season run. The interactive infographic was created in flash and can be viewed on flash enabled devices here.
