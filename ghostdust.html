Ghostdust Audio Visual Project
context:
Web Development 100 Class proj. 2015


Ghostdust was a web page assignment created to illustrate the students basic web development skills. The student was assigned to create three html pages that utilize basic CSS layout principles that can be navigated using a ul and nav powered navigation bar. The images below may be clicked or rolled over to be mocked up. This project is currently live and can be visited here.
